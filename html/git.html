Git

  • git status — отображает актуальное состояние файлов в рабочем каталоге и индексе, включая изменённые, добавленные и новые файлы.
  • git add файл — выбирает конкретный файл для включения в будущий коммит, добавляя его в индекс.
  • git commit -m «Описание изменений» — формирует новый коммит с комментарием, описывающим суть внесённых правок.
  • git log — отображает список коммитов в порядке от самых новых к более старым, позволяя проследить историю изменений проекта.
  • git push — внесение изменений на удаленный репозиторий
  • git pull — вносит изменения из удаленного репозитория в локальный.
